a webgl tool for exploring strange attractors and 3d dynamical systems. the kind of side project that started as "i want to see what happens if…" and ended up as a real piece of software that other people use.

- threlte + three.js, runge-kutta integration on the gpu
- hand-rolled equation parser so users can type their own systems
- the ui is a love letter to the 70s scientific calculator
